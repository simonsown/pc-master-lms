═══════════════════════════════════════════════
2. SO SÁNH GIÁ LINH KIỆN (Chống Chặt Chém)
═══════════════════════════════════════════════
- Nhập tay tên linh kiện + giá hóa đơn
- So khớp với 127 linh kiện thật từ GearVN
- Hiển thị: giá hóa đơn vs giá thị trường
- Trạng thái: Hợp lý / Hơi đắt / Chặt chém!
- % chênh lệch + thanh bar trực quan
- Đánh giá AI + lời khuyên chi tiết
- So sánh tổng tiền (invoice vs market)
- URL: /builder/invoice-check

═══════════════════════════════════════════════
3. CAREER BUILD — GỢI Ý CẤU HÌNH THEO ƯỚC MƠ
═══════════════════════════════════════════════
- Chọn nghề nghiệp (AI Engineer, 3D Designer, Video Editor, Streamer, Coder, Office, Mac...)
- Hoặc nhập ước mơ tự do (có spell check)
- Thêm yêu cầu đặc biệt (ngân sách, RGB, AMD...)
- AI gợi ý cấu hình chi tiết từng linh kiện
- Mỗi linh kiện có: tên, loại, giá, LÝ DO chọn, LINK MUA GEARVN (hoa hồng)
- Hỗ trợ cả Windows PC và Mac (MacBook/Mac Studio)
- Chia tab Windows / Mac khi có cả hai
- Click "Lắp ráp" để build ngay
- URL: /builder/career-build

- Mỗi máy: specs, giá, lý do phù hợp, link mua GearVN

═══════════════════════════════════════════════

═══════════════════════════════════════════════
CÔNG NGHỆ SỬ DỤNG
═══════════════════════════════════════════════
- Frontend: Next.js 16, React, Three.js (R3F)
- Backend: Next.js API routes, Supabase (PostgreSQL)
- AI: Groq API, Gemini API
- Xác thực: Supabase Auth + Google OAuth
- Hand Tracking: MediaPipe Hands + Face Landmarker
- 3D: React Three Fiber (R3F)
- Deploy: Vercel (production)
- CSS: Inline styles, CSS variables (dark/light theme)
